# PDIR Workflow - Starter Kit

Template para criar aplicativos fullstack Next.js com IA usando o Método PDIR.

## Início Rápido

```bash
git clone git@github.com:hmaurus/pdir-workflow.git seu-projeto
cd seu-projeto
/configurar-projeto
```

## Documentação

- **[Manual Completo](docs/manuais/starterkit-pdir.md)** - Starter kit + Método PDIR
- **[Comandos Claude Code](.claude/README.md)** - Referência rápida
- **[Regras do Projeto](CLAUDE.md)** - Stack, padrões, arquitetura

## Comandos Principais

```bash
/configurar-projeto                     # Wizard inicial
/dev-start                              # Inicia desenvolvimento
/pdir:pdir-criar-issue [descrição]      # Cria Issue
/pdir:pdir-implementar-tarefa [issue]   # Implementa
/pdir:pdir-commit                       # Commit + push
/pdir:pdir-merge-tarefa [pr]            # Merge + limpeza
```
